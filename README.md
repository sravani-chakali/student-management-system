# Student Management System

## 📌 Project Overview
This is a simple Student Management System built using Python Flask. It helps to manage student records like adding, editing, viewing, and deleting students.

## ⚙️ Technologies Used
- Python
- Flask
- HTML
- SQLite

## 🚀 Features
- Add student
- View students
- Edit student
- Delete student
